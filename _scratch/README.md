# _scratch/ — Agent 用スクラッチ / 一時受け渡しフォルダ

このリポジトリ専用のローカル作業領域です. **system の `/tmp` とは別物**で, ユーザが Agent に
渡す資料 (PDF・CSV・スクリーンショット等) や, コードの実行・一時生成物をここに置きます.

- **Hakyll の build 対象外**なので**サイトに公開されません** (旧 `tmp/` → 一時 `files/` を経て改称.
  `files/` は `src/Main.hs` の `files/**` ルールで公開されてしまうため不適でした).
- `README.md` 以外の中身は `.gitignore` で commit 対象外です.

なお `files/` は将来「公開してよいダウンロードファイル」を置く枠として `files/**` ルールごと
残してあります (現状は空). **公開したくない作業ファイルは必ず `_scratch/` に置いてください.**
